# FaceLock Project
얼굴인식 + 블록체인 도어락 시스템

## 아키텍처
- Frontend: React (사용자 UI)
- Backend: Node.js (API & 인증 처리)
- Device: Raspberry Pi + Python (얼굴인식 & 도어락 제어)
- Blockchain: Solidity (출입 기록 저장)
